# Interstellar

### Next level Laravel Admin panel

#### Requirements
1. Laravel 9
2. PHP 8.1

#### Installation guide
1. add the below lines in your `composer.json file`
    ```
    "repositories": [
            {
                "type": "vcs",
                "url": "https://github.com/sohag-pro/interstellar.git"
            }
        ]
    ```
2. run `composer require sohag-pro/interstellar`
